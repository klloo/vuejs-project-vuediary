## Vue Diary
#### vue.js 학습 용도로 제작한 간단한 다이어리 웹 서비스
#### 구현 기능
- **로그인 화면** : 최초 서비스 접근 시 사용자 이름을 입력할 수 있는 로그인 화면 제공
- **일기 목록** : 일기 목록을 조회(타임라인, 그리드 형식 제공)
- **일기 상세 정보** : 일기 목록에서 일기를 클릭하면 해당 일기의 상세 내용을 조회한다.
- **일기 작성 및 수정** : 일기를 작성한다. (제목, 날씨, 기분, 내용)
  
#### 사용 기술 스택
- [x] Vue
- [x] Vuex
- [x] Vue Router
- [x] JavaScript 
      
#### 실행 방법
```
yarn install
yarn serve
```
